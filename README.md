# Logic Circuit Design Projects ⚡

This repository contains two major digital logic design projects developed during the **CMPE243 Logic Circuit Design** course at Kadir Has University. The projects demonstrate the ability to design, simplify, and implement both Combinational and Sequential logic circuits.

## 🚌 Project 1: Smart Bus Route Selector (Combinational Logic)
A combinational digital system that automates the conversion of a 4-bit Bus ID into a 4-bit destination Stop Code. 
* **Methodology:** Truth tables, Minterms extraction, Karnaugh Map (K-Map) simplification.
* **Simulation:** Validated using MATLAB/Simulink logic gates.
* **Hardware Implementation:** Physically built on a CADET breadboard using standard digital ICs (7404 NOT, 7408 AND, 7432 OR).

## 🏭 Project 2: Factory Production Control (Sequential Logic / FSM)
A sequential control system designed to manage the operation order of multiple factory machines (M1 → M5 → M3 → M6 → M7) ensuring one process finishes before the next begins.
* **Methodology:** Modeled as a **Mealy Finite State Machine (FSM)**.
* **Components:** D Flip-Flops are used for state storage. Next-state and output equations were minimized using K-Maps.
* **Simulation:** Fully implemented and verified using MATLAB/Simulink and **Stateflow**.

### 🛠️ Tools & Technologies
* MATLAB / Simulink / Stateflow
* Boolean Algebra & K-Map Minimization
* Breadboard prototyping & IC Wiring
